# Column exists

检查表指定的字段是否存在 (结果字段名是 Boolean 类型)

# 数据库查询

数据库里面的左连接。左连接就是两张表执行左关联查询，把左边的表数据全部查询出来

# 数据库连接

可以执行两个数据库的查询，和单参数的表输入

# 检查表是否存在

和 Column exists 一样，只不过这个检查表是否存在

# 流查询

查询前把数据都加载到内存中，并且只能进行等值查询
